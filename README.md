
# 🚢 Titanic Data Analysis

## 📌 Project Overview

This project explores the Titanic dataset to analyse survival patterns based on different features such as gender, class, age, and fare. The goal is to extract meaningful insights using Python and data analysis techniques.

---

## 🛠 Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook

---

## 📊 Key Analysis Performed

* Data cleaning and handling missing values
* Exploratory Data Analysis (EDA)
* Survival rate comparison by gender and class
* Correlation between fare and survival
* Age distribution analysis

---

## 📈 Key Insights

* Female passengers had ~74% survival rate compared to ~19% for males
* First-class passengers had significantly higher survival rates
* Higher fare-paying passengers were more likely to survive
* Younger passengers showed slightly higher survival trends

---

This project demonstrates the ability to clean, analyse, and interpret real-world data. It highlights how socio-economic factors influenced survival outcomes and showcases core data analysis skills.
