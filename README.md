## 📊 Titanic Data Analysis

### 📌 Project Overview
This project analyses the Titanic dataset to uncover patterns affecting passenger survival. The goal is to extract meaningful insights using Python and data analysis techniques.

---

### 🎯 Objectives
- Identify key factors influencing survival
- Perform data cleaning and preprocessing
- Explore relationships between variables
- Generate insights using visualisations

---

### 🛠 Tools & Technologies
- Python (Pandas, NumPy)
- Matplotlib / Seaborn
- Jupyter Notebook

---

### 🔍 Key Analysis Performed
- Data cleaning (handling missing values)
- Exploratory Data Analysis (EDA)
- Survival rate analysis by gender, class, and age
- Feature relationships and correlations

---

### 📈 Key Insights
- Female passengers had higher survival rates
- First-class passengers were more likely to survive
- Age and fare showed moderate influence on survival

---

### 📂 Project Structure
- `titanic_data_analysis.ipynb` → main analysis notebook
- `README.md` → project documentation
